# Headlines

Für Headlines nutzen wir die **Futura Bold BT** also Futura Bold BT (Schriftstärke 700).
Farbe ist abhängig von der Nutzungsart.

```type
{
  "headings": [46,40,26],
  "font": "Futura Md Bt",
  "color": "#333333",
  "weight": 700,
}
```


# Fließtext

Für Fließtext nutzen wir die **Futura Lt BT** also Futura Light BT (Schriftstärke 400).
Farbe ist abhängig von der Nutzungsart.

```type
{
  "paragraphs": ["20/28"],
  "font": "Futura Lt Bt",
  "color": "#333333",
}
```
